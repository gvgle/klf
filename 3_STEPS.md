# 🎯 只需 3 步解决白屏

## 问题

pvz.guagle.com 还是白屏，因为 GitHub Pages 没有开启。

## 解决方法（3 步）

### 第 1 步
在浏览器打开这个网址：
```
https://github.com/gvgle/klf/settings/pages
```

### 第 2 步
找到一个写着 "Branch" 的下拉菜单，点击它，选择 **gh-pages**

### 第 3 步
点击 **Save** 按钮
